## STEPS TO RUN LOCALLY

1. Run the main application

```node 
npm run dev
```

2. Run the Campaign Scheduler worker in a newly created terminal

```node
npm run scheduler
```

3. Run the Campaign Orchestrator worker in a newly created terminal

```node
npm run orchestrator
```

4. Run the  Email Router worker in a newly created terminal

```node
npm run router
```
5. Run the Email Sender worker in a newly created terminal

```node
npm run sender
```

6. Run the Email Verifier worker in a newly created terminal
   
```node
npm run verifier
```

1. Lastly, run the Reply Ingestion worker in a newly created terminal

```node
npm run replier
```
